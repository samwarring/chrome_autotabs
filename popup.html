Tab Harmony

Sort Tabs

Keep tabs sorted alphabetically by their reversed domain name.

A tab's reversed domain name is its domain name in reverse order, excluding the final ".com".

For example, the reversed domain name of mail.google.com is google.mail.

Group Tabs

Automatically group tabs if they share a domain suffix.

The domain suffix is the last part of the website name, excluding the final ".com". The tab group will be named after the domain suffix.

For example, mail.google.com and calendar.google.com share the domain suffix google.

Tab Group Size

0

Minimum number of tabs required for a group.

When a tab group falls below this many tabs, it will be automatically ungrouped.
